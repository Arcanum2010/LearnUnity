    Unity project examples for the book Learn Unity 4 for iOS Game Development

    Project maintained by technicat Hosted on GitHub Pages — Theme by mattgraham

    This is the github site for an upcoming book, Learn Unity 4 for iOS Game Development, currently available as an Apress alpha book.

    The github projects don't include assets from the Unity Asset Store and Standard Assets that are used in the book, but the HyperBowl-style bowling game developed by following the book, complete with Asset Store assets and Unity Standard Assets, is on the App Store.

    The book and app demonstrate Unity graphics, audio and physics, scripting in JavaScript (and a little bit of C#), touch screen and accelerometer input, iAd banner and interstitial ads, Game Center leaderboard and achievements, complete bowling rules, and utilize free Asset Store packages:

    • Eighteen Free Substances from Allegorithmic
    • Free ArtskillZ Texture Pack 01 from ArtskillZ
    • Skeletons Pack from bisaniyehocam
    • General Music Set from Gianmarco Leone
    • Free SFX Package from Bleep Blop Audio
    • Barrel from Universal Image

    Questions can be posted on the Unity forum. You may also be interested in Sue Blackman's Beginning 3D Game Development with Unity, also from Apress.

    Technical review provided by March Shaerer of Gayasoft.
