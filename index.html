This is the GitHub page for the upcoming book, Learn Unity 4 for iOS Game Development.

The GitHub projects, built up cumulatively through each chapter, do not include assets from the Unity Asset Store and Standard Assets that are used in the book.

Content

The book and app (see below) demonstrate Unity graphics, audio and physics, scripting in JavaScript (and a little bit of C#), touchscreen and accelerometer input, iAd banner and interstitial ads, Game Center leaderboard and achievements, complete rules for a bowling game, and utilizes free Asset Store packages.

The book also covers debugging, profiling and optimization, testing with the Unity Remote and iOS Simulator, build and App Store submission procedures.

Corrections and Clarifications

Retail Description

  • The retail description lists the Unity's Mecanim animation system. The book does mention but does not use Mecanim. Instead, a chapter is devoted to a dancing skeleton example using Unity's Legacy animation system and a free skeleton package on the Asset Store.
  • The retail description lists Unity's support for gyroscope on iOS devices. The book does mention but does not use Unity's gyroscope API (Input.gyroscope). The book does describe how to access the touchscreen (Input.touches), accelerometer (Input.accelerometer), and webcam (WebCamTexture).

Chapter 1

  • Unity iOS Basic is now free and no longer $400 (yay!) same for Unity Android Basic, by the way.

Chapter 2

  • Chapter 2 has a mangled phrase like "function is called a Log.DebugLog". It should be "called the function Debug.Log"
